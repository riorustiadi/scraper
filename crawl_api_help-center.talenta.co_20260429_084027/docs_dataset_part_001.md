Talenta LMS (Learning Management System) adalah platform pembelajaran digital yang membantu perusahaan dalam merencanakan, mengelola, serta mengevaluasi program pelatihan karyawan secara terstruktur.

Melalui Talenta LMS, Anda dapat membuat dan mengelola *course*, menyusun materi pembelajaran, menentukan tingkat kesulitan pelatihan, serta memantau progress dan hasil pembelajaran karyawan secara terpusat.

Untuk mengakses Talenta LMS, Anda dapat masuk ke akun Talenta Anda. Kemudian klik **“HRIS”** dan pilih **LMS App** untuk masuk ke halaman utama LMS untuk employee (students) atau pilih **LMS Backyard** untuk masuk ke halaman pengelolaan LMS untuk administrator.

## A. LMS App

Pada LMS App, employee (students) dapat mengakses dan melihat daftar *course* yang telah dibuat oleh administrator. Melalui LMS App, employee juga dapat melihat sertifikat yang mereka miliki dan *course* yang diikuti.

## B. LMS Backyard

LMS Backyard merupakan halaman khusus yang digunakan oleh administrator untuk mengelola seluruh aktivitas pembelajaran di Talenta LMS. Melalui LMS Backyard, administrator dapat membuat, mengatur, dan menerbitkan *course*, serta mengelola seluruh komponen pembelajaran seperti kategori, level, konten, hingga *assessment.*

Pada halaman LMS Backyard, terdapat beberapa menu utama yang dapat Anda gunakan, yaitu:

* [**Courses**](https://help-center.talenta.co/hc/id/articles/50166612266009), untuk membuat dan mengelola course pelatihan yang didalamnya juga terdapat submenu seperti:

  + **Course list**, untuk melihat daftar course yang sudah dibuat.
  + **Contents**, untuk mengunggah dan mengelola materi pembelajaran.
  + **Categories**, untuk mengelompokkan course berdasarkan tema dan subtema.
  + **Levels**, untuk menentukan tingkat kesulitan *course.*
  + **Assessments**, untuk membuat evaluasi seperti *quiz, pre-test,* dan *post-test.*
* **Settings**, menu untuk [mengundang pengguna](https://help-center.talenta.co/hc/id/articles/50768908382617) dan melihat kuota pengguna yang dimiliki.

Pada LMS Backyard, Anda juga dapat mengundang karyawan untuk mengakses LMS dengan role employee atau administrator.

Melalui **Mekari Talenta Recruitment**, selain [tes kepribadian MBTI](https://help-center.talenta.co/hc/id/articles/54018820786969), kini Anda dapat menyusun dan mengelola tes penilaian kandidat lainnya sendiri. Tes ini dapat Anda buat dengan tipe jawaban pilihan ganda, jawaban teks panjang, maupun numerik. Dengan fitur ini, rekruter dapat dengan lebih mudah mengirimkan tes dan memonitor hasilnya melalui satu pintu saja, yaitu Mekari Talenta Recruitment.

Ikuti langkah-langkah berikut untuk membuat dan mengelola tes penilaian yang disediakan di halaman **Assessments**.

## A. Membuat Tes Penilaian untuk Kandidat

1. Masuk ke menu **Assessments**.
2. Klik **“Create assessment”** untuk membuat tes penilaian baru.
3. Maka, Anda diarahkan ke halaman **Create Assessment**. Pada tahap pertama, Anda diminta untuk melengkapi **Assessment informations** yang berupa informasi untuk kandidat mengenai tes ini. Berikut penjelasannya.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1. | Assessment name | Isi dengan nama tes penilaian. |
| 2. | Description | Isi dengan deskripsi tes penilaian ini. |
| 3. | Duration to complete assessment | Isi dengan durasi yang diberikan kepada kandidat untuk menyelesaikan penilaian ini dalam hitungan menit. |
| 4. | Measured capabilities | Isi dengan kemampuan spesifik yang dinilai dalam tes ini dengan jumlah maksimal 10 (sepuluh) kemampuan. |
| 5. | Assessment instruction | Isi dengan instruksi khusus untuk kandidat dalam memahami prosedur pengerjaan tes. |

4. Jika sudah lengkap, klik **“Continue”.**
5. Selanjutnya, pada tahap kedua, yaitu **Add question**, Anda dapat menambahkan pertanyaan atau soal dengan klik **“Create Question”.** Maka, akan muncul beberapa opsi tipe pertanyaan atau soal untuk Anda pilih.

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
   | 1. | Multiple choice | Pilih tipe pertanyaan ini untuk menyusun soal tipe pilihan ganda. Jika Anda memilih opsi ini, maka muncul *sidebar* yang dapat Anda lengkapi dengan contoh dan penjelasan seperti berikut.  **1. Question name:** Masukan pertanyaan dari soal ini.  **2. Answer type:** Pilih salah satu tipe jawaban berdasarkan jumlah benarnya. **Single correct answer** berarti satu jawaban saja yang benar, sedangkan **Multiple correct answer** berarti jawaban benar lebih dari satu.  **3. Options:** Lengkapi opsi jawaban pilihan ganda kemudian klik ikon **“bulat**” untuk memilih jawaban yang benar.  **4. Delete:** Klik ikon ini untuk menghapus opsi jawaban.  **5. Add option:** Klik untuk menambah opsi jawaban.  **6. Create another question:** Centang ini untuk segera membuat pertanyaan pilihan ganda lain setelah menyimpan pertanyaan ini.  **7. Save:** Klik untuk menyimpan soal. |
   | 2. | Essay | Pilih tipe pertanyaan ini untuk menyusun soal tipe jawaban panjang. Jika Anda memilih opsi ini, maka muncul *sidebar* yang dapat Anda lengkapi dengan contoh dan penjelasan seperti berikut.  **1. Add option:** Masukkan pertanyaan dari soal ini.  **2. Create another question:** Centang ini untuk segera membuat pertanyaan dengan jawaban panjang lain setelah menyimpan pertanyaan ini.  **3. Save:** Klik untuk menyimpan soal. |
   | 3. | Number | Pilih tipe pertanyaan ini untuk menyusun soal tipe jawaban yang hanya dapat dijawab dengan angka saja. Jika Anda memilih opsi ini, maka muncul *sidebar* yang dapat Anda lengkapi dengan contoh dan penjelasan seperti berikut.  **1. Question name:** Masukkan pertanyaan dari soal ini. **2. Right answer:** Masukkan jawaban yang benar (hanya dalam bentuk angka).  **3. Create another question:** Centang ini untuk segera membuat pertanyaan dengan jawaban panjang lain setelah menyimpan pertanyaan ini.  **4. Save:** Klik untuk menyimpan soal. |
6. Berikut adalah contoh tampilan soal yang telah selesai disusun. Anda dapat melihat soal beserta tipe pertanyaannya pada masing-masing tabel soal. Pada tabel, Anda dapat mengurutkan kembali rangkaian soal dengan menarik dan melepaskan kolomnya (*drag and drop*). Klik ikon **“pensil”** untuk mengubah soal dan jawabannya. Untuk menghapus, klik **“-”**. Selain itu, Anda dapat klik **“Shuffle question”** untuk mengacak soal pada setiap kandidat yang dikirimkan.

**Penting**  
Seperti pada contoh di atas, pada satu tes penilaian, bisa berupa berbagai macam tipe soal. Anda juga dapat membuat tes penilaian Anda dengan satu tipe soal saja.

6. Apabila soal sudah terangkai dengan sesuai, klik **“Publish”** untuk menyimpan soal yang siap untuk langsung diberikan kepada kandidat. Atau, apabila Anda masih ingin memodifikasi soal dan belum ingin menerbitkan, klik **“Save as draft”** untuk kembali menyelesaikannya lain waktu.
7. Maka, tes penilaian berhasil dibuat dan tersimpan pada daftar **Assessments**.

## B. Mengelola Tes Penilaian untuk Kandidat yang telah Dibuat

1. Pada halaman Assessments, Anda dapat memfilter daftar tes penilaian berdasarkan statusnya dengan klik **“All status”**. Selain itu, terdapat filter tanggal terakhir soal termodifikasi pada **“Select date”**. Kemudian, untuk mempermudah Anda mencari soal, Anda juga dapat mengetikkan nama pada **“Select assessment name”** atau yang juga biasa disebut sebagai **Search bar**.
2. Untuk mengubah rincian atau memodifikasi soal tertentu, klik **“Actions”** lalu **Edit**.

   Maka, Anda akan diarahkan ke tahap **Assessment informations** dan **Add questions** di mana Anda dapat memodifikasi soal tersebut dan mempublikasikannya kembali. Tahap ini mengikuti alur dari pembuatan tes penilaian yang dapat Anda baca di [bagian A.](#h_01KNKE2XZ5ZB8Z6ABMK55Q3WN3)
3. Sedangkan, untuk membatalkan publikasi soal, klik **“Actions”** lalu **Unpublish**. Tindakan ini akan membuat tes penilaian tidak dapat dikirimkan ke kandidat.
4. Pada *pop-up* yang muncul, konfirmasi tindakan Anda dengan klik **“Unpublished”**.
5. Tindakan lainnya pada halaman **Assessments** adalah penghapusan tes penilaian pada daftar. Ada dapat melakukannya dengan klik **“Actions”** lalu **Delete**.
6. Pada *pop-up* yang muncul, konfirmasi tindakan Anda dengan klik **“Delete assessment”**.

Demikian panduan membuat, menyusun, dan mengelola tes penilaian untuk kandidat dalam proses rekrutmen. Selanjutnya, Anda dapat mempelajari cara mengelola susunan jadwal pada Calendar, [di sini.](https://help-center.talenta.co/hc/id/articles/42469173015193)

Melalui [Mekari Talenta Performance Management](https://help-center.talenta.co/hc/id/articles/11427496661785), Anda dapat mengakses halaman atas profil karyawan secara terpusat untuk mengambil keputusan strategis terkait promosi dan *succession planning* dengan lebih objektif berdasarkan data historis performa yang komprehensif.

Berikut penjelasannya.

1. Masuk ke akun [Mekari Talenta Performance Management](http://review.talenta.co) Anda.
2. Pergi ke menu **“Talent Directory”**.
3. Maka, Anda diarahkan ke halaman berikut. Anda dapat menerapkan filter pada daftar yang ditampilkan. Berikut penjelasannya.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1. | Branch | Filter daftar berdasarkan cabang karyawan. |
| 2. | Organization | Filter daftar berdasarkan organisasi karyawan. |
| 3. | All filters | Klik untuk memfilter data secara lebih lengkap, seperti **Job level, Job position, Employment status**, dan **SBU.** Setelah filter sesuai, klik **“Apply filter”.**  1. **Ikon panah atas/bawah:** Untuk menutup/membuka *checkbox filter.* 2. **Search bar:** Untuk mencari nama-nama tertentu pada kategori filter yang Anda pilih. Contohnya, nama cabang, organisasi, dan lainnya. 3. **Checkbox:** Untuk mencentang nama kriteria yang ingin Anda filter. 4. **Add filter:** Untuk menambahkan kategori filter lainnya, seperti **Job level, Job position, Employment status**, dan **SBU**. |
| 4. | Search bar | Gunakan fitur ini untuk mencari nama karyawan dengan mengetikkan kata kunci di sini. |

4. Selain itu, Anda juga dapat melihat profil pegawai dengan klik **“View talent profile”**.
5. Maka, Anda diarahkan ke halaman **Talent profile** pegawai tersebut.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1. | View [Employee name] profile on Talenta | Klik untuk diarahkan ke halaman [General info pada Employee profile](https://help-center.talenta.co/hc/id/articles/9285010107417) karyawan. |
| 2. | Save to PDF | Klik untuk mengunduh *file PDF* **Talent profile** karyawan ini. |

6. Anda dapat melihat rincian terkait performa karyawan, seperti **“Profile”**. Di sini, Anda dapat melihat informasi seperti **General information, Employment information,** dan **Talent mapping.**
7. Rincian lainnya terkait performa karyawan adalah **“Performance”**. Di sini, Anda dapat melihat informasi seperti **performance score, performance history** beserta grafiknya.
8. Rincian selanjutnya adalah **“Competencies”**. Di sini, Anda dapat melihat rincian nilai **Competency** karyawan. Anda dapat klik ikon **“tanda panah ke bawah”** untuk memilih nama *competency.*
9. Rincian terakhir adalah **“History”**. Di sini, Anda dapat melihat **Transfer history, Reprimand history,** dan **Learning & development history.**

Demikian panduan memonitor Talent profile pada Talent directory pada Mekari Talenta Performance Management. Selanjutnya, Anda dapat mempelajari mengenai cara mengelola potensi karyawan pada perusahaan Anda, seperti Succession Planning [di sini.](https://help-center.talenta.co/hc/id/articles/12371107543193)
