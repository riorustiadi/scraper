Fitur Calendar pada [New Talenta Recruitment](https://help-center.talenta.co/hc/id/articles/34567830638361) memungkinkan HR untuk mengkoordinasikan jadwal wawancara dan acara-acara penting dengan lebih mudah. Berikut langkah-langkah cara mengelola fitur calendar new Talenta recruitment.

1. Pergi ke <https://recruitment-new.talenta.co/>.
2. Pilih menu **Calendar**.
3. Maka, Anda diarahkan ke halaman berikut.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1 | Page indicator | Anda dapat klik agar kalender mengarahkan Anda ke tampilan rentang waktu **saat ini.** |
| 2 | Previous/Next | Klik untuk melihat halaman kalender dalam tampilan mingguan sebelumnya atau setelahnya. |
| 3 | Calendar filter | Klik untuk memfilter kalender yang ditampilkan berdasarkan **jenis acara terkait rekrutmen**, seperti interview, offer, dan lain sebagainya. |
| 4 | More | * Apabila halaman kalender menampilkan **rentang waktu per bulan**, Anda dapat klik **“More”** untuk menampilkan acara pada hari tersebut. * Apabila halaman kalender menampilkan **rentang waktu per hari**, Anda dapat klik **“More”** untuk menampilkan acara pada jam di hari tersebut. |

4. Untuk melihat rincian acara terkait interview atau wawancara calon kandidat, klik pada salah satu acara.
5. Maka, muncul tampilan rincian acara tersebut seperti berikut. Anda dapat mengelola jadwal tersebut dengan klik ikon **“titik tiga”.**  
     

   **- View candidate profile:** Klik untuk melihat profil kandidat yang diwawancara.**- Edit interview schedule:** Klik untuk mengubah jadwal wawancara.**- Cancel interview schedule:** Klik untuk membatalkan jadwal wawancara.**- Submit interview feedback:** Klik untuk membuat feedback dari wawancara.

Demikian panduan cara mengelola calendar new Talenta recruitment. Selanjutnya, Anda dapat mempelajari cara mengelola schedule interview pada new Talenta recruitment, [di sini.](https://help-center.talenta.co/hc/id/articles/34781370644761)
