| 5. | **Search** | Klik untuk mencari nama employee tertentu dengan mengetikkan kata kuncinya. |

Anda dapat memantau kehadiran karyawan melalui fitur Smart Attendance melalui Mekari Talenta. Dengan fitur ini, Anda akan mendapatkan notifikasi terkait kehadiran karyawan seperti saat karyawan terlambat, pulang lebih cepat atau tidak hadir. Sehingga, Anda dapat memantau kehadiran karyawan secara praktis melalui notifikasi-notifikasi tersebut.

**Penting**  
Untuk menggunakan fitur ini, pastikan Anda telah mengajukan Smart Attendance Notification. Pelajari caranya [di sini.](https://help-center.talenta.co/hc/id/articles/32572684071321)

Berikut adalah langkah-langkah mengatur workflow fitur Smart Attendance.

1. Masuk ke menu **Settings**.
2. Klik **Workflows,** kemudian pilih **Attendance.**
3. Maka, Anda akan diarahkan ke halaman ini. Anda dapat melihat sisa kuota workflows di bagian yang di-*highlight* berikut.  
     

   Anda dapat menambahkan kuota workflow Attendance dengan klik tulisan biru **“Request quota”** atau **“Request workflow”.** Maka, muncul pop-up berikut. Anda dapat melengkapi **data Anda** pada form tersebut sebelum mengirimkannya dengan klik **“Submit request”**.
4. Pada bagian ini, Anda dapat memilih template-template tertentu. Untuk saat ini, Anda dapat menggunakan template **Inform Late In Daily** dan **Early Out Daily** saja. Klik salah satu **“Use template”** untuk menggunakan template workflow tersebut.
5. Berikut adalah contoh halaman form template workflow. Anda dapat mengedit kolom-kolom yang tersedia.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Kolom/Tombol** | **Penjelasan** |
| 1 | Workflow name | Ubah template nama workflow apabila diperlukan. |
| 2 | Description | Ubah template deskripsi workflow apabila diperlukan. |
| 3 | Trigger name | Ubah nama trigger sesuai pada opsi yang disediakan, yaitu: - Employee clocks in late - Employee clocks out early |
| 4 | There are no specific criteria, it applies to all employees | Centang opsi ini apabila notifikasi Smart Attendance ini akan Anda dapatkan dari seluruh karyawan. Apabila Anda tidak mencentangnya, maka Anda dapat memilih kriteria karyawan tertentu seperti berikut.  Anda dapat memilih salah satu kriteria karyawan, seperti **Branch, Organization, Job position, dan lain sebagainya**. Contoh di atas merupakan kriteria **berdasarkan Job position**. Karena memilih kriteria tersebut, Anda dapat memilih beberapa **kriteria spesifik**nya, seperti Staff IT dan Staff lainnya. Anda dapat menambahkan kriteria lainnya dengan klik **“Add criteria”**. |

6. Pada bagian Actions, Anda dapat mendefinisikan pengaturan untuk kriteria yang telah Anda tentukan sebelumnya.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Kolom/Tombol** | **Penjelasan** |
| 1 | Action name | - **Action Name:** Pilih tindakan yang akan dilakukan workflow saat dipicu. Saat ini, hanya tersedia **Send notification.**   Kemudian, Anda dapat memilih metode yang tersedia.   - **Push notification on Talenta app**: Centang untuk mengirim pemberitahuan push ke perangkat seluler karyawan melalui aplikasi Talenta.  **- Talenta inbox:** Centang untuk mengirim pemberitahuan ke kotak masuk karyawan dalam Talenta.  **- Email notification:** Centang untuk mengirim pemberitahuan email ke penerima yang ditentukan. |
| 2 | Send notification to | Kolom ini untuk mencentang penerima notifikasi. Workflow dapat Anda atur sedemikian rupa untuk memberi tahu:   **- Employee:** Karyawan yang sesuai.  **- Employee's manager:** Manajer langsung karyawan.  **- Employee's approval line:** Lajur persetujuan karyawan di atas tersebut.  **- Notify Others:** Karyawan tertentu yang perlu diberi tahu. |
| 3 | Add employee | Jika Anda mencentang **Notify others**, maka Anda dapat memilih **nama karyawan** pada sidebar yang muncul. Kemudian, klik **“Submit”.** |
| 4 | Notification frequency | Pilih seberapa sering notifikasi dikirim, yaitu: - Monthly - atau, Weekly   Jika Anda telah berlangganan Smart Attendance, maka terdapat 3 (tiga) opsi lainnya, yaitu: - Daily - Shift start - Shift end |
| 5 | Notification weekly on | Apabila Anda memilih **Monthly**, maka tentukan **tanggalnya** di kolom ini. Sedangkan, apabila Anda memilih **Weekly**, Anda dapat menentukan **harinya** di kolom ini. |

7. Klik **“Save”** untuk menyimpan workflow.
8. Maka, respon Anda berhasil diajukan.
9. Anda dapat memantau status pengajuan workflow Anda di kolom **Workflows - Attendance** seperti berikut.

Demikian cara mengatur workflow smart attendance. Selanjutnya, apabila Anda ingin mengetahui cara mengatur notifikasi time management, Anda dapat klik [di sini.](https://help-center.talenta.co/hc/id/articles/37233395182105)
