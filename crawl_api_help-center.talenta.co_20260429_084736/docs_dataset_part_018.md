title: Bagaimana Cara Mengelola Succession Plan pada Talenta Performance
article_id: 12371107543193
section_id: 8966183392281

category_name: Performance Management

Succession Plan pada [Talenta Performance](https://help-center.talenta.co/hc/id/articles/11427496661785) dapat digunakan untuk mengidentifikasi karyawan yang berpotensi untuk menempati key position di masa depan dan mengelompokkannya ke dalam succession pool .

Berikut langkah-langkah untuk membuat sucession plan:

1. Masuk ke akun Talenta Performance Anda dan pilih menu**“Succession plan”**.
2. Maka, Anda akan diarahkan ke halaman berikut.  
   Keterangan:  

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Kolom/Tombol** | **Deskripsi** |
   | 1 | Create succession plan | Tombol untuk membuat succession plan baru. |
   | 2 | Filter | Manfaatkan filter berdasarkan divisi (Organization) dan klik **“Apply Filter”** untuk menerapkan filter. |
   | 3 | Key position/Employee | Ubah tampilan daftar berdasarkan: * **Key position:** Berisi daftar succession pool berdasarkan Key Position yang telah dibuat beserta organization dan successor talentnya. * **Employee:** Berisi daftar karyawan yang sudah diidentifikasi sebagai karyawan berpotensi untuk masing-masing key position. |
   | 4 | Search bar | Manfaatkan search bar untuk mencari kata kunci tertentu berdasarkan daftar. |
   | 5 | View detail | Klik untuk melihat halaman rincian succession plan.  Tombol ini berubah menjadi tombol ****“Action”**** apabila Anda memilih tampilan daftar Employee. Tombol tersebut memiliki fungsi yang berbeda dengan ****“View detail”**.** Terus baca panduan untuk mengetahuinya. |
3. Maka, untuk membuat succession plan yang baru, klik **“Create Succession plan”**.
4. Anda akan diarahkan ke halaman Create Succession Plan - Employment criteria seperti berikut.  
     
   Keterangan:  

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Kolom/Tombol** | **Deskripsi** |
   | 1 | Key position | Tentukan key position untuk succession pool yg akan dibuat. Daftar key position yang ada di sini sesuai dengan job position Talenta. |
   | 2 | Organization | Pilih organisasi yang sesuai dengan Key Position. |
   | 3 | Service length | Masukkan masa jabatan karyawan jika diperlukan sebagai syarat identifikasi karyawan sebagai successor. |
   | 4 | Employee status | Pilih status karyawan yang berada pada perusahaan apakah permanen, kontrak, freelance atau lainnya jika status karyawan diperlukan sebagai syarat  identifikasi karyawan sebagai successor. |
5. Jika kolom - kolom telah diisi dengan benar dan lengkap, klik **“Next”.** Anda akan diarahkan ke halaman selanjutnya, yaitu Standard competencies seperti berikut.  
     
   1. Pilih tipe assessment yang Anda inginkan. Anda dapat memilih **Talenta performance** jika assessment dilakukan menggunakan cycle di dalam Talenta performance, atau **Manual input** jika Anda menggunakan pihak ketiga untuk melakukan assessment terhadap successor.  

      Competency yang ada di dalam pilihan dapat ditambahkan, diubah atau dikurangi melalui ****“Competency item”**** yang ada di menu [Set up Competency item](https://help-center.talenta.co/hc/id/articles/17247609720089).
   2. Klik Group name yang tersedia, lalu **“Next”.**
6. Pada halaman selanjutnya yaitu Standard competencies, klik **“Add talent”** untuk menambahkan karyawan yang berpotensi menjadi successor untuk key position.
7. Klik tanda **“+”** untuk menambahkan karyawan dan tanda **“-”** untuk menghapus karyawan. Dan klik **“Add”** untuk menyimpan.
8. Anda dapat klik **“Filter”** untuk mencari karyawan berdasarkan **filter branch**, **organization**, **job level**, **job position**,**employee status,** dan **service length**.
9. Selanjutnya, Anda dapat menentukan **Readiness level** (opsional) yang didasarkan dari hasil assessment. Anda dapat mengosongkan terlebih dahulu dan mengubahnya nanti di halaman succession pool detail ketika sudah melihat gap hasil assessment untuk setiap successor.
10. Maka, hasil Succession pool yang telah Anda buat terlihat seperti berikut. Klik **“View detail”** untuk melihat secara detil.
11. Jika key position yang ditentukan vacant dan ada successor yang sudah memenuhi standard competency yang ditentukan, Anda dapat mempromosikannya pada posisi tersebut dengan klik **“Action”** dan pilih **“Promote”**.  

    Jika Anda sudah mengkonfirmasi action promote untuk succession terkait, sistem akan mengirimkan notifikasi ke Super Admin Default Talenta melalui inbox Talenta dan email. Selanjutnya Super Admin Talenta dapat melanjutkan proses employee transfer dengan menekan tombol proceed promote yang ada di inbox Talenta ataupun email.
12. Lalu, akan muncul pop up konfirmasi, klik **“Promote”** untuk melanjutkan.
13. Selain itu, pada **“Action”**, Anda dapat mengubah readiness level dengan klik **“Edit readiness”**, melihat hasil perbandingan antara standard competency yang ditentukan dengan hasil assessment yang sudah dilakukan untuk successor dengan klik **“View detail”**, dan hapus karyawan dengan klik **“Remove from pool”**.
14. Klik **“Edit”** untuk mengubah succession pool yang telah Anda buat.
15. Atau Anda juga dapat menambahkan karyawan dengan klik**“Add Successor Talent”**.

Demikian adalah cara mengelola succession plan pada Talenta Performance. Selanjutnya, Anda dapat mempelajari cara set up competency pada Talenta Performance [di sini.](https://help-center.talenta.co/hc/id/articles/17247609720089)

title: Bagaimana Cara Melakukan Assign Reimbursement (Tampilan Baru)

section_id: 8907431596057

category_id: 8889743635225

Anda dapat memberlakukan [kebijakan reimbursement](https://help-center.talenta.co/hc/id/articles/12245991430681) terhadap karyawan tertentu melalui menu **Finance** submenu Reimbursement. Aktivitas ini hanya dapat dilakukan oleh user dengan roles Superadmin dan admin.

 Berikut langkah-langkahnya:

1. Masuk ke menu **Finance,** pilih **Benefit** **Reimbursement**.
2. Klik **Assign or Update**.
3. Klik tombol **“Actions”** kemudian **Assign reimbursement.**
4. Pilih kebijakan reimbursement yang ingin diberlakukan dengan klik **“Reimbursement name”***.* Kemudian, isi kolom **Description** jika diperlukan.
5. Klik **“Select employee”** untuk memilih karyawan.  
     

   **Penting**  
   Anda dapat menginput nama-nama karyawan secara massal untuk menghemat waktu. Pelajari cara impor dan ekspor employee untuk reimbursement, [di sini.](https://help-center.talenta.co/hc/id/articles/12248238496153)
6. Pilih **nama-nama karyawan** pada daftar **bagian kiri**. Nama yang terpilih akan **berpindah ke bagian kanan**. Klik **“Save”** jika nama karyawan yang terpilih sudah sesuai.
7. Pastikan nama-nama karyawan sudah sesuai. Untuk menghapus satuan, Anda dapat klik ikon **“-”**. Sedangkan, untuk menghapus nama secara massal, Anda dapat klik **checkbox** di sisi kiri nama karyawan. Klik **“Save**” untuk assign reimbursement ke karyawan dengan nama-nama yang terdaftar.
8. Maka, assign reimbursement tersimpan dan Anda diarahkan ke halaman rinciannya seperti berikut.
9. Klik **“Assign or update”.**
10. Anda dapat melihat tindakan *reimbursement* yang di-*assign* tersebut pada tab **Transaction History.** Klik **“Action”** untuk melihat melihat rinciannya **(View detail)** atau melakukan **Edit**.

Demikian penjelasan mengenai cara melakukan assign reimbursement. Selain melakukan assign, roles Superadmin/ Admin juga dapat melakukan pengajuan Reimbursement, lihat [di sini](https://help-center.talenta.co/hc/id/articles/12247764796953) untuk mempelajarinya lebih lanjut.
