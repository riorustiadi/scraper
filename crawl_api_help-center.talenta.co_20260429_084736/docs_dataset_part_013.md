Melalui fitur On-call management, Anda dapat merencanakan karyawan mana yang harus siaga pada tanggal yang dipilih dan mengirimkan notifikasi kepada orang tersebut ketika terjadi keadaan darurat. Sistem ini terintegrasi dengan fitur [Live attendance](https://help-center.talenta.co/hc/id/articles/11229563878553) dan [Payroll](https://help-center.talenta.co/hc/id/articles/11165497432601) yang secara otomatis menghitung jadwal/shift On-call untuk memberikan kompensasi ketika orang tersebut datang bekerja di luar jam kerjanya.

**Penting**  
- Pastikan perusahaan sudah berlangganan add-on On-call. Untuk menambahkan fitur ini, hubungi kami di [support-hr@mekari.com](mailto:support-hr@mekari.com).  
- Perbarui aplikasi Talenta Anda ke ADR (2.xx.0), iOS (2.xx.0) atau lebih baru untuk memastikan manajemen panggilan berfungsi di aplikasi Talenta.  
- Pastikan Anda telah mengatur akses role untuk mengakses halaman On-call management. Pelajari panduan mengatur akses role [di sini.](https://help-center.talenta.co/hc/id/articles/11607064557849)

Sebelum membuat On-call schedule/shift baru, Anda dapat mempelajari sekilas tentang halaman On-call Index berikut.

1. Masuk ke menu "**Time",** kemudian klik **On call.**
2. Kemudian, pilih halaman **On-call schedule.** Berikut adalah penjelasannya.  
     
   Keterangan:

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
   | 1 | Help | Klik untuk membuka panduan ini. |
   | 2 | Import on-call | Klik untuk membuat *on-call schedule* secara massal dengan mengunduh dan mengunggah *spreadsheet.* |
   | 3 | Create on-call | Klik untuk membuat *on-call schedule* secara satuan. |
   | 4 | Filter periode waktu | Klik untuk memilih jangka waktu data schedule yang tertera dalam daftar. |
   | 5 | All filters | Klik untuk memfilter data schedule berdasarkan tipe tertentu, seperti **Branch, Organization, Job position, Job level, atau Employment status.** |
   | 6 | Adjust column | Klik untuk menyesuaikan kolom yang ditampilkan pada daftar. |
   | 7 | Search bar | Klik untuk mencari data karyawan berdasarkan nama atau IDnya. |
   | 8 | Checkboxes | Klik checkbox di samping nama karyawan untuk melakukan tindakan massal **Cancel on-call.** |
   | 9 | Shift detail | Arahkan kursor ke sini untuk memunculkan rincian shift. |
   | 10 | Actions | Klik untuk melakukan tindakan **Assign on-call shift** (Menugaskan on-call shift)**, Remove on-call shift** (Menghapus on-call shift), atau **Cancel on-call** (Membatalkan on-call shift).  - Jika **belum melakukan Assign on-call shift**, maka Anda **tidak dapat melakukan Remove on-call shift.** - Jika belum melakukan **Remove on-call shift** sebelum **Cancel on-call,** maka Anda **tidak dapat melakukan Cancel on-call.** |
3. Sedangkan berikut adalah tampilan index **On-call activity** dan penjelasannya.  
     
   Keterangan:

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
   | 1 | Help | Klik untuk membuka panduan ini. |
   | 2 | Import on-call activity | Klik untuk mengimpor data *spreadsheet* aktivitas *on-call.* |
   | 3 | Filter waktu | Klik untuk memilih waktu data aktivitas karyawan yang tertera dalam daftar. |
   | 4 | All filters | Klik untuk memfilter data aktivitas karyawan berdasarkan tipe tertentu, seperti **Branch, Organization, Job position, Job level,** atau **Employment status.** |
   | 5 | Adjust column | Klik untuk menyesuaikan kolom yang ditampilkan pada daftar. |
   | 6 | Download | Klik untuk mengunduh data aktivitas yang ditampilkan di daftar ini ke dalam bentuk *spreadsheet.* |
   | 7 | Search bar | Klik untuk mencari data karyawan berdasarkan nama atau IDnya. |
   | 8 | Checkboxes | Klik checkbox di samping nama karyawan untuk melakukan tindakan massal **Delete activity.** |
   | 9 | Actions | Klik untuk melakukan tindakan **View history log** (Melihat riwayat)**, Edit activity** (Mengubah)**, Delete activity** (Menghapus), atau **Remove on-call shift**(Penugasan ang telah ditetapkan akan dihapus & karyawan tidak akan memiliki jadwal siaga) aktivitas on-call karyawan tersebut.  Data yang dapat diubah ***(Edit)*** adalah data **attendance, seperti clock-in, clock-out, break start,** dan **break end.** |

Demikian sekilas tentang halaman on-call index pada menu Time Management. Selanjutnya, Anda dapat mempelajari [cara membuat on-call schedule dan shift.](https://help-center.talenta.co/hc/id/articles/23493034991385)

Pada Talenta, kini Anda dapat dengan mudah mengumpulkan data [karyawan yang baru](https://help-center.talenta.co/hc/id/articles/20747239794713) bergabung atau *new joiner*. Data yang terkumpul tersebut akan secara otomatis masuk ke dalam database Talenta. Namun, Anda perlu mengatur form submission sebelum dikirimkan ke *new joiner*. Berikut langkah-langkahnya.

1. Masuk ke menu **"Employees"** lalu pilih **New joiner submission**.
2. Kemudian, Anda akan melihat daftar *New joiner submission* di halaman ini. Dan untuk mengatur form submission klik tombol **“Form Settings”**.
3. Data *new joiner* terbagi atas beberapa bagian. Bagian-bagian yang perlu diisi oleh new joiner tersebut dapat Anda kustomisasi. Berikut penjelasannya.

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Bagian** | **Deskripsi** |
   | 1 | Preview form | Klik untuk melihat *preview form* dari perspektif new joiner. |
   | 2 | Personal data | Anda dapat memilih untuk menampilkan kolom **Mobile phone, Place of birth, Gender, Blood type,** dan **Phone** dengan klik *toggle button*-nya.  *Toggle* berwarna abu-abu tua bersifat default atau data wajib yang tidak dapat Anda ubah. |
   | 3 | Identity & address | Anda dapat memilih untuk menampilkan kolom **NIK (NPWP 16 Digit), Postal code, Residential address, Passport number and expire date,** dan **Citizen ID address** dengan klik *toggle button*-nya. |
   | 4 | Bank account | Anda dapat memilih untuk menampilkan kolom **Bank name, Account holder name,** dan **Account number** dengan klik *toggle button*-nya. |
   | 5 | Tax type | Anda dapat memilih untuk menampilkan kolom **NPWP 15 digit** dan **PTKP status** dengan klik *toggle button*-nya. |
   | 6 | Education & work experience | Anda dapat memilih untuk menampilkan kolom **Formal education, Work experience,** dan **Informal education** dengan klik *toggle button*-nya. |
   | 7 | Additional document | Klik **Automatically sync attachments to employee file** agar sistem secara otomatis melakukan sinkronisasi lampiran yang disertakan di form dengan data employee file Mekari Talenta.  Anda dapat memilih lampiran apa yang harus diunggah oleh *new joiner* pada kolom **”Select file category”**. Apabila lampiran tersebut wajib diunggah oleh *new joiner*, maka Anda dapat mencentang **“Required to fill by new joiners”.** Untuk menambahkan lampiran lainnya yang dapat diisi oleh *new joiner*, maka  klik **“Add additional document”**.  Data **additional document** merupakan file category yang tersimpan pada pengaturan di halaman berikut <https://hr.talenta.co/setting/company-information#myFilesTab>. |
4. Dan klik **“Save”** untuk menyimpan pengaturan.

Demikian penjelasan mengenai cara mengatur forms settings untuk *new joiner submission*. Selanjutnya, Anda dapat mempelajari bagaimana cara mengumpulkan data *new joine*r, [di sini](https://help-center.talenta.co/hc/id/articles/20747239794713).

**Onboarding** adalah fitur untuk mengelola serangkaian kegiatan yang ditugaskan kepada karyawan yang baru saja bergabung dalam perusahaan. Setelah [karyawan baru diundang ke Talenta](https://help-center.talenta.co/hc/id/articles/11263500976153), data - data karyawan tersebut masuk ke halaman ini. Sehingga, Anda dapat mengelola segala proses Onboarding karyawan tersebut di sini.

**Penting**- Tampilan pada panduan ini merupakan tampilan terbaru fitur Onboarding. **Tampilan lama fitur ini akan segera tidak berlaku lagi *(deprecated).*** Selain itu, fitur ini hanya tersedia pada paket **Enterprise/Plus/360**. Hubungi tim support kami di [info-hr@mekari.com](mailto:info-hr@mekari.com) untuk upgrade paket Anda atau untuk mendapatkan informasi lebih lanjut.  
- Sementara itu, Anda dapat mempelajari tampilan baru Onboarding pada panduan ini atau melihat **video Onboarding terbaru [di sini.](https://youtu.be/fXJSsZb4BC0)**

Berikut adalah sekilas tentang halaman Onboarding pada Talenta.

1. Masuk ke menu **Employees** lalu pilih **Onboarding**.
2. Maka, Anda akan diarahkan ke tampilan berikut.  
     
   ****Keterangan:****

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
   | 1 | Settings | Jika Anda perlu melakukan pengaturan Onboarding pada Settings, klik tombol ini. |
   | 2 | Filter | Kolom ini berfungsi untuk memfilter daftar nama karyawan yang baru yang Onboarding. Anda dapat memfilter berdasarkan **Job position** dan **Onboarding status.** |
   | 3 | Guidebook shortcut | Tombol ini akan mengarahkan Anda ke guidebook ini. |
   | 4 | Search bar | Ketikkan kata kunci tertentu untuk mencari data employee pada daftar Onboarding. |
   | 5 | Checkbox | Centang pada checkbox apabila Anda ingin melakukan tindakan massal. |
   | 6 | Actions | Klik tombol ini untuk melakukan tindakan tertentu pada karyawan tersebut. Anda dapat **melakukan** **Start onboard** pada **status Ready on board** atau **Complete onboarding** pada **status In progress.**Selain itu, Anda juga dapat **melihat Detail Onboarding** pada **kedua status tersebut.** |
   | 7 | Details | Klik tombol ini untuk membuka halaman details. |

Demikian sekilas penjelasan tentang halaman Onboarding pada menu Company di Talenta.
