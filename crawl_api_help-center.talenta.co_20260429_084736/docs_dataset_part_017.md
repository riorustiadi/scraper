Pada halaman New & Active (Phase 1) [Talenta Insight](https://help.center.talenta.co/hc/id/articles/9362092632217), Anda dapat melihat informasi mengenai karyawan yang berada di dalam perusahaan seperti jumlah karyawan yang aktif, karyawan baru, transfer karyawan, retention rate, Average tenure hingga Ratio HR to employee. Anda juga dapat melihat laporan Headcount Trend dan Breakdown yang tersaji dalam bentuk grafik.

Berikut langkah-langkah untuk mengakses informasi New & Active pada Talenta Insight.

1. Klik **"Insight"** pada bagian Applications di Menu Dashboard. Atau pergi ke [insights.talenta.co](https://www.talenta.co/insight-talenta/).
2. Klik menu **“Headcount”** pada bagian atas lalu pilih **New & active**.
3. Pada bagian atas, Anda dapat melihat filter yang dapat Anda gunakan untuk memfilter data pada bagian Headcount Trend dan Breakdown. Anda dapat memilih filter berdasarkan rentang waktu, status karyawan, Branch, Organization, Job Level dan Job Position lalu klik **“Apply filter”**.
4. Selanjutnya, Anda dapat melihat data informasi karyawan.  
     

   |  |  |  |
   | --- | --- | --- |
   | **No** | **Kolom** | **Penjelasan** |
   | 1 | All active employee | Menampilkan informasi karyawan yang aktif. |
   | 2 | New hire | Menampilkan informasi karyawan baru. Jika Anda mengkliknya, maka Anda dapat menemukan detil karyawan baru serta mengunduh data tersebut. |
   | 3 | Employee transfer | Menampilkan informasi transfer karyawan. Jika Anda mengkliknya, maka Anda dapat menemukan detil karyawan serta mengunduh data tersebut. |
   | 4 | Retention rate | Menampilkan matriks yang mengukur persentase karyawan aktif di perusahaan pada periode tertentu.  Dengan formula seperti berikut: **- Total Last Day Employee** = Ambil total karyawan dengan kondisi tanggal join (join\_date) <= tanggal akhir bulan sekarang. **- Total First Day Employee** = Ambil total karyawan dengan kondisi tanggal join (join\_date) <= tanggal sekarang. **- Total New Hire Employee** = Ambil total karyawan dengan kondisi bulan dari tanggal join = bulan dari tanggal sekarang. **- Total Retained** = Total Last Day Employee - Total New Hire Employee. **- Retention Rate** = Total Retained/Total First Day Employee. |
   | 5 | Average tenure | Menampilkan data rata-rata karyawan bekerja di perusahaan.  Dengan formula seperti berikut: **- List Employee Max Tenure** = Ambil nilai Tenure terbesar dengan kondisi bulan resign >= bulan sekarang. **- Total Average Tenure** = Ambil rata-rata dari List Employee Max Tenure (Average). |
   | 6 | Ratio HR to employee | Estimasi perbandingan kuantitatif antara HR dengan karyawan berdasarkan job title (HR, HC, GA, SDM, Human, General Affair, People, Sumber Daya Manusia, Payroll, Recruitment, Benefit, Compensation dan Company Benefit). |
   | 7 | Export history icon | Untuk melihat data new hire dan employee transfer yang di ekspor. |
5. Kemudian, Anda dapat melihat grafik Headcount trend yang berisikan informasi karyawan aktif, karyawan saat ini dan karyawan baru.  
     

   |  |  |  |
   | --- | --- | --- |
   | **No** | **Kolom** | **Penjelasan** |
   | 1 | Show value number | Centang untuk menampilkan informasi angka pada grafik. |
   | 2 | Download | Klik untuk mengunduh data dalam format CSV. |
   | 3 | All active employee | Menampilkan grafik data karyawan aktif per bulan ini dengan garis berwarna biru. Anda juga dapat mengklik current employee dan new hire jika ingin data all active employee saja yang muncul. |
   | 4 | Current employee | Menampilkan data jumlah karyawan per bulan ini dengan garis berwarna hijau. Anda juga dapat mengklik all active employee dan new hire jika ingin data current employee saja yang muncul. |
   | 5 | New hire | Menampilkan data jumlah karyawan yang baru bergabung di bulan ini dengan garis berwarna ungu. Anda juga dapat mengklik all active employee dan current jika ingin data new hire saja yang muncul. |
6. Terakhir, terdapat grafik Headcount breakdown yang menampilkan jumlah karyawan berdasarkan Employment Data (Branch, Organization, Job Position, Job Level, Grade, Employment Status dan Service Length) atau Demografi (Jenis Kelamin, Usia, Pendidikan Terakhir, Agama, dan PTKP Status).  
     

   |  |  |  |
   | --- | --- | --- |
   | **No** | **Kolom** | **Penjelasan** |
   | 1 | Compare with last period | Centang untuk membandingkan data saat ini dengan periode sebelumnya. |
   | 2 | Search keyword | Kolom untuk melakukan pencarian berdasarkan keyword tertentu. |
   | 3 | Download | Klik untuk mengunduh data dalam format CSV. |
   | 4 | Breakdown by | Kolom untuk memfilter data berdasarkan:  **Employment Data:** 1 .Branch 2. Organization 3. Job Position 4. Job Level 5. Grade 6. Employment Status 7. Service Length  **Demografi:** 1. Jenis Kelamin  Bagian ini menampilkan tambahan persentase seperti:  - Female supervisor: Persentase jumlah supervisor wanita di perusahaan. - Male supervisor: Persentase jumlah supervisor pria di perusahaan. - Ratio male to female: Rasio perbandingan karyawan perempuan dan laki-laki di perusahaan.  2. Usia  Bagian ini menampilkan tambahan persentase seperti:  - Millenials: Persentase karyawan di perusahaan yang lahir tahun 1995 - 2012. - Generation Z: Persentase karyawan di perusahaan yang lahir tahun 1980 - 1994. - Generation X: Persentase karyawan di perusahaan yang lahir tahun 1965 - 1979. - Baby boomer: Persentase karyawan di perusahaan yang lahir tahun 1946 - 1964.  3. Pendidikan Terakhir 4. Agama 5. PTKP Status |

   Demikian penjelasan mengenai New & Active. Selanjutnya, Anda dapat mempelajari mengenai Headcount pada Talenta Insight, [di sini](https://help-center.talenta.co/hc/id/articles/9367355974809).

Fitur Manpower Planning berguna untuk mengoptimalkan sumber daya manusia di sebuah perusahaan. Dengan adanya fitur ini, perusahaan dapat [menetapkan rencana jumlah karyawan](https://help-center.talenta.co/hc/id/articles/15785442658201) setiap bulannya serta menemukan karyawan pengganti ketika terdapat karyawan yang resign. Untuk mengakses fitur ini, Anda harus memiliki role **Super Admin** atau **Admin**.

Pada tab Overview, Anda dapat melihat ringkasan mengenai manpower planning yang sudah diatur. Untuk lebih jelasnya, berikut sekilas mengenai tab Overview pada Manpower Planning Talenta. 

|  |  |  |
| --- | --- | --- |
| **No** | **Menu** | **Deskripsi** |
| 1 | Manage Planning | Untuk membuat rencana pengelolaan sumber daya manusia di perusahaan. |
| 2 | Request for this Month | Untuk melihat total permintaan berdasarkan bulan secara otomatis. |
| 3 | Need Approval | Untuk melihat request jumlah karyawan yang membutuhkan persetujuan. |
| 4 | Data per Tahun | Untuk mencari data berdasarkan tahun. |
| 5 | Filter | Untuk memfilter berdasarkan **Job Position** dan **Employment Status.** |
| 6 | Job Position | Posisi pekerjaan. |
| 7 | Detail Position | Untuk melihat detail posisi dari jabatan **Permanent, Contract, Probation.** |
| 8 | Current Headcount | Jumlah karyawan di perusahaan pada saat ini  Data di dalamnya bisa dipengaruhi oleh adanya **karyawan yang resign**, **transfer karyawan**, **penambahan karyawan baru** dan **request penambahan karyawan baru.** |
| 9 | Planned Headcount | Jumlah karyawan di perusahaan yang telah direncanakan. |
| 10 | Export | Untuk mengekspor data pada tab Overview. |
| 11 | Guidebook | Untuk melihat guidebook yang berkaitan dengan fitur Manpower Planning. |

Demikian penjelasan tentang tab Overview pada Manpower Planning. Klik [di sini](https://help-center.talenta.co/hc/id/articles/15785492836761) untuk melakukan pengajuan headcount pada Manpower Planning.

Pada Talenta Performance, Superadmin dapat membuat, melihat rincian, dan menduplikasi template yang disajikan dalam bentuk list. Template perlu dibuat sebelum [membuat cycle.](https://help-center.talenta.co/hc/id/articles/13972050822297)  
Pastikan Anda sudah memahami panduan berikut untuk memulai Review Cycle.

Berikut adalah overview halaman template settings pada Talenta Performance.

1. Masuk ke submenu Template pada menu **Settings**.
2. Maka, halaman Template Settings terlihat seperti berikut.  
     
   **Keterangan:**  

   |  |  |  |
   | --- | --- | --- |
   | **No.** | **Nama Kolom/Tombol** | **Deskripsi** |
   | 1. | Add new template | Tombol ini berfungsi untuk membuat template baru. Anda dapat mengetahui cara membuat template baru di sini. |
   | 2. | Search bar | Anda dapat mengetikkan keyword template yang akan Anda cari di kolom ini. Lalu, klik **“Enter”** pada keyboard Anda. |
   | 3. | Activation toggle button | Activation toggle button berfungsi untuk mengaktifkan atau menonaktifkan template tersebut. |
   | 4. | Three dots | Tombol ikon ini menyediakan fungsi **Edit, View,** dan **Duplicate** template yang berstatus **Active**.  Sedangkan, untuk yang berstatus **Non-active**, maka hanya tersedia pilihan **View** dan **Duplicate.** |

Demikian adalah overview halaman template settings pada Talenta Performance. Selanjutnya, Anda dapat mempelajari tentang Bagaimana Cara Mengelola Template pada Talenta Performance [di sini.](https://help-center.talenta.co/hc/id/articles/12333746961561)
