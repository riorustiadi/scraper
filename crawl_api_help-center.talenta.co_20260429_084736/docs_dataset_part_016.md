Talenta memiliki fitur **Offboarding** yang mana mempermudah proses pengelolaan dan pemantauan proses resign karyawan perusahaan Anda. Dengan Offboarding, karyawan dapat mengajukan resign dengan melalui proses approval. Anda juga dapat memanfaatkan fitur [resign karyawan secara satuan](https://help-center.talenta.co/hc/id/articles/9132852241049) ataupun [massal.](https://help-center.talenta.co/hc/id/articles/11149104362009)
