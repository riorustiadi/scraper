Melalui Talenta, Anda dapat membuat dan mengelola review performa karyawan dan pribadi melalui Review Cycle. Melalui fitur ini, Anda dapat membuat cycle review baru dengan 3 (tiga) tujuan cycle yang berbeda, yaitu Performance, Competency, dan Evaluation Review.

Berikut adalah penjelasan singkat tentang halaman index review cycle yang menyimpan riwayat review yang pernah dibuat.

1. Log in ke [review.talenta.co](https://review.talenta.co/login?referrer=%2Fhome).
2. Klik menu **Review Cycle**.
3. Maka, Anda akan diarahkan ke halaman berikut.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1 | Help | Klik untuk mendapatkan bantuan melalui Guidebook atau Quick tour. |
| 2 | Template setting | Tombol ini merupakan shortcut untuk menuju halaman pengaturan template. Pelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/13972653920153) |
| 3 | Create new cycle | Klik untuk membuat siklus review baru.  Masing - masing pilihan memiliki fungsi tersendiri, yaitu:  * **Performance review:** Siklus review yang dapat digunakan untuk melakukan penilaian kinerja karyawan berdasarkan KPI/goal dan indikator kualitatif lainnya seperti company values, soft competency, dll. Pelajari cara membuat review ini [di sini.](https://help-center.talenta.co/hc/id/articles/11428359433241)  * **Competency review:** Siklus review yang dapat digunakan untuk melakukan assessment kompetensi karyawan untuk mengetahui kelebihan dan kekurangan karyawan untuk posisinya yang sekarang atau masa depan.. Pelajari cara membuat review ini [di sini.](https://help-center.talenta.co/hc/id/articles/20793430931737) * **Evaluation review:** Siklus review yang dapat digunakan untuk mengevaluasi kinerja karyawan pada masa kerja tertentu, seperti masa probation atau kontrak. Pelajari cara membuat review ini [di sini.](https://help-center.talenta.co/hc/id/articles/20793498345369) |
| 4 | Filter cycle | Klik untuk memfilter daftar siklus review yang pernah dibuat. |
| 5 | Open approval setting | Klik untuk membuka pengaturan *approval review*. Pelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/11371231766425) |
| 6 | Search bar | Ketikkan kata kunci di sini untuk mencari data tertentu pada daftar review yang pernah dibuat. |
| 7 | Total cycle | Pada baris ini, Anda dapat melihat jumlah timeframe di dalam siklus berdasarkan status publishnya |
| 8 | Actions | Klik untuk memunculkan opsi tindakan - tindakan tertentu pada review cycle yang pernah dibuat.   1. Opsi tindakan pada review dengan siklus tidak otomatis. 2. Opsi tindakan pada siklus review dengan tipe evaluation.   **- View detail:** Klik untuk melihat rincian cycle tersebut. Pada detail tersebut, Anda juga dapat mengunduh hasil review yang telah selesai. Pelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/11463635215001) **- Update repeat date cycle:** Klik untuk memperbarui siklus berulang secara manual. **- Start automatic cycle:** Klik untuk membuat siklus evaluation mengelompokkan karyawan probation/contract secara otomatis **- Lock/unlock review result:** Klik untuk mengunci atau membuka kunci hasil review dimana hasil review akan dapat di-edit reviewer atau tidak. **- Delete:** Klik untuk menghapus review.   **Penting** Untuk mempelajari lebih lanjut tentang cara melakukan tindakan - tindakan yang tersedia, Anda dapat mempelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/8966186583833) |

Demikian sekilas tentang halaman review cycle pada Talenta Performance. Selanjutnya, dapat mempelajari tentang cara melihat hasil review secara rinci untuk masing-masing role, [di sini.](https://help-center.talenta.co/hc/id/articles/11464683644825)

Anda dapat mengelola goals pada yang dapat diakses pada [Talenta Performance.](https://help-center.talenta.co/hc/id/articles/11427496661785) Terdapat 4 tipe goal yang dapat Anda sesuaikan, yaitu Goal Individual, Team, Organization, dan Company.

Untuk mempelajari cara mengelola halaman Index Goals pada Talenta Performance, Anda dapat mengikuti langkah-langkah berikut.

1. Login ke [Akun Talenta Performance Review](https://review.talenta.co/login?referrer%3D%252Fconsultant&sa=D&source=docs&ust=1720405532247925&usg=AOvVaw1jJhYUDAD4h_JftpSUZo90) Anda.
2. Pilih Goals dan Anda dapat pilih tipe keperluan goals yang ingin Anda buat, yaitu Individual, Team, Organization, atau Company.
3. Maka, Anda akan diarahkan ke halaman Index Goals. Di sini, Anda dapat melakukan berbagai macam tindakan untuk mengelola goal. Berikut merupakan contoh tampilan goal tipe Individual.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1 | Help | Klik untuk mendapatkan bantuan berupa halaman guidebook atau fitur quick tour. |
| 2 | Upload new goal | Klik untuk mengunggah goal baru. Pelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/31855390011673) |
| 3 | Create new goal | Klik untuk membuat goal baru. Pelajari panduannya [di sini.](https://help-center.talenta.co/hc/id/articles/9853528944665) |

4. Anda dapat melihat index masing-masing tab **On going, Closed, dan Requested**.  
     

   **- Ongoing:** Tampilan daftar goal yang sedang berjalan.**- Closed:** Tampilan daftar goal yang sudah selesai dan berhenti berjalan.**- Requested:** Tampilan daftar goal yang sedang dalam proses pengajuan approval. Tab ini muncul apabila Anda merupakan approval line dari goal. Pelajari cara membuat approval goal [di sini.](https://help-center.talenta.co/hc/id/articles/11585622071705)
5. Berikut adalah tampilan index pada tab **Ongoing** yang fiturnya **sama** seperti tab **Closed.**

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1 | Filter employee | Klik untuk memfilter daftar goal  berdasarkan nama karyawan tertentu sebagai goal owner. |
| 2 | Open approval setting | Klik untuk membuka tampilan shortcut ke ESS Setting untuk menentukan settingan approval untuk goal. |
| 3 | Search bar | Anda dapat memanfaatkan kolom pencarian untuk mencari kata kunci tertentu pada daftar. |
| 4 | Checkbox | Klik untuk memunculkan tombol **“Actions”** dimana Anda dapat melakukan tindakan massal tertentu.  Pada nama-nama goal yang dicentang, Anda dapat memilih untuk melakukan:  * Bulk update: Update massal goal. * Bulk edit: Edit massal goal. * Bulk close: Tutup massal goal. * Delete: Hapus massal goal. |
| 5 | Goal progress | Arahkan kursor ke atas bar goal progress dan ikon **“Update progress”** akan muncul. Pelajari cara mengupdate goal progress [di sini.](https://help-center.talenta.co/hc/id/articles/9855001509145) |
| 6 | Goal history | Anda dapat melihat pop-up riwayat tindakan yang telah dilakukan pada goal tersebut. |
| 7 | Titik tiga | Klik untuk memunculkan opsi tindakan-tindakan tertentu.   - **View parent goal:** Klik untuk melihat parent goal dari goal ini. - **Align goal:** Klik untuk menyambungkan antara child goal dengan parent goal. Klik [di sini](https://help-center.talenta.co/hc/id/articles/31987438652697) untuk mempelajari panduannya. - **Update progress:** Klik untuk mengupdate progress goal. - **View detail:** Klik untuk melihat rincian goal. - **Close goal:** Klik untuk menutup dan menghentikan goal. - **Delete goal:** Klik untuk menghapus goal. |

6. Berikut adalah tampilan index pada tab **Requested**.

|  |  |  |
| --- | --- | --- |
| **No.** | **Nama Tombol/Kolom** | **Deskripsi** |
| 1 | Filter employee | Klik untuk memfilter daftar berdasarkan nama karyawan tertentu. |
| 2 | Filter request type | Klik untuk memfilter request goal berdasarkan tipe pengajuan approval All request type, Create new goal, Edit goal, atau Update goal. |
| 3 | Open approval setting | Klik untuk membuka tampilan shortcut ke ESS Setting. |
| 4 | Search bar | Anda dapat memanfaatkan kolom pencarian untuk mencari kata kunci tertentu pada daftar. |
| 5 | Actions | Klik untuk memunculkan opsi tindakan pada requested goal, yaitu **View detail** (melihat rincian), **Approve/reject** (shortcut ke halaman inbox untuk approve/reject request), atau **Cancel request** (membatalkan pengajuan).   Anda hanya dapat melakukan **View detail** pada goal dengan status **Rejected.** |

Demikian sekilas tentang halaman index goals pada Talenta Performance. Selanjutnya, Anda dapat mempelajari tentang goal setting [di sini.](https://help-center.talenta.co/hc/id/articles/19375121224857)

**Candidate** merupakan salah satu menu pada New Talenta Recruitment yang digunakan untuk menampilkan tahapan-tahapan rekrutmen dan memproses pelamar pada setiap tahapan yang pernah dibuat. Berikut merupakan tampilan fitur-fitur yang terdapat pada menu **Candidate**.

|  |  |  |
| --- | --- | --- |
| **No** | **Kolom** | **Penjelasan** |
| 1 | Bulk update | Klik untuk melakukan update data kandidat secara massal dengan menggunakan Excel. |
| 2 | Add candidate | Tombol untuk menambahkan kandidat baru. |
| 3 | Filter | Fitur yang dapat memudahkan Anda dalam melakukan pencarian data kandidat secara lebih detail melalui kolom-kolom yang dapat Anda filter dengan kategori tertentu. |
| 4 | Manage column | Merupakan fitur yang memudahkan Anda untuk menyaring dan mengelola data tabel informasi terkait tahapan perekrutan. |
| 5 | Search column | Kolom yang dapat Anda gunakan untuk melakukan pencarian berdasarkan kata kunci tertentu. |
| 6 | Pipeline stages | Menunjukkan jumlah kandidat dalam kategori-kategori tertentu. Pipeline ini memudahkan Anda dalam melakukan penyaringan data berdasarkan tahapan rekrutmen yang sudah dibuat. Sebagai contoh, apabila Anda ingin melihat data kandidat yang telah berada di tahap **“Interview”** maka Anda dapat klik filter **“Interviews”** dan tampilan berikut akan muncul. |
| 7 | Tabel Informasi | Menampilkan detail informasi terkait kategori pipeline yang Anda pilih.  - Apabila Anda merupakan pelanggan **Ultimate Hire**, Anda dapat melihat kolom Candidate score yang menunjukkan kecocokan kriteria, profil, dan skill kandidat untuk posisi pekerjaan yang Anda *post*. - Anda dapat melihat tagging kandidat yang telah di-*restrict* dalam proses rekrutmen di tabel informasi seperti berikut. |
| 8 | Move to | Tombol untuk memindahkan kandidat ke stage tertentu sesuai dengan pengaturan dan sisa kuota Anda. |
| 9 | Ikon “titik tiga” | Tombol untuk mengelola informasi rekrutmen pada setiap data kandidat yang tertera pada tabel.    **Keterangan:** - **View details:** Untuk melihat informasi terkait **Candidate** secara detail. - **Edit candidate**: Untuk melakukan perubahan data kandidat. **- Move to another job listing:** Untuk memindahkan kandidat pada lowongan kerja lainnya. - **Not interested:** Tidak tertarik untuk melanjutkan proses perekrutan kandidat terkait. Dalam hal ini kandidat tidak berhasil lolos tahapan rekrutmen. - **Archived:** [Mengarsipkan informasi kandidat](https://help-center.talenta.co/hc/id/articles/34774798206361). - **Mark as spam:** Menandai informasi kandidat sebagai spam. - **Delete:** Menghapus data kandidat. - **Block (block candidate for app):** Memblokir data kandidat. - **Mark as withdrawn**: Kandidat mengundurkan diri. |

Demikian adalah sekilas tentang menu Candidate pada Talenta Recruitment. Selanjutnya, Anda dapat mempelajari tentang cara menambahkan kandidat [di sini](https://help-center.talenta.co/hc/id/articles/34772410147097-Bagaimana-Cara-Add-Candidate-pada-New-Talenta-Recruitment).

**Job Listings** merupakan menu pada [New Talenta Recruitment](https://help-center.talenta.co/hc/id/articles/34567069229465-Sekilas-tentang-New-Talenta-Recruitment) yang digunakan untuk membuat dan mengelola tahapan rekrutmen pekerjaan yang nantinya akan diikuti oleh para kandidat karyawan.

Berikut ini penjelasan sekilas tampilan menu **Job Listings**:

|  |  |  |
| --- | --- | --- |
| **No.** | **Kolom** | **Deskripsi** |
| 1. | **Import** | Klik untuk secara langsung mengunggah Job listing dalam jumlah banyak atau secara massal. |
| 2. | **Create job listing** | Tombol untuk menambahkan lowongan pekerjaan baru. |
| 3. | **Filters** | Fitur yang dapat memudahkan Anda dalam melakukan pencarian data **Job listing** secara lebih detail melalui kolom-kolom yang dapat Anda filter secara **By jobs**, **By publications**, dan **By hiring team**.  Filter **By jobs** terdiri dari kolom-kolom berikut:  - **Work location:** Filter berdasarkan lokasi kerja. - **Organization:** Filter berdasarkan nama organisasi. - **Employment status:** Filter berdasarkan status karyawan. - **Working type:** Filter berdasarkan tipe pekerjaan yang tersedia.  Filter berdasarkan **By publications** terdiri dari kolom-kolom berikut:  - **Publication date:** Tanggal pengunggahan rekrutmen kerja. - **Closing date:** Batas akhir rekrutmen kerja. - **Publication platform:** Sumber/lokasi pengunggahan rekrutmen. - **Creator:** Peran yang membuat rekrutmen.  Filter **By hiring team** terdiri dari kolom berikut: |
| 4. | **Filters untuk “Publication status”** | Filter yang akan memudahkan Anda dalam melakukan penyaringan data berdasarkan **Publication status** rekrutmen yang sudah dibuat.  Sebagai contoh, apabila Anda ingin melihat data yang telah berhasil terunggah, maka Anda dapat klik filter **“Published”**. Lalu akan  muncul tampilan berikut. |
| 5. | **Filter untuk mengelola tabel informasi data** | Merupakan fitur yang memudahkan Anda untuk menyaring dan mengelola data tabel informasi perekrutan. |
| 6. | **Tabel Informasi** | Menampilkan detail informasi terkait rekrutmen karyawan baru. |
| 7. | **Ikon “titik tiga”** | Tombol untuk melakukan tindakan-tindakan berikut pada job listing pada setiap data yang tertera pada tabel. |

Demikian sekilas menu *job listings* pada New Talenta Recruitment. Selanjutnya pelajari terkait [cara membuat job listing pada New Talenta Recruitment](https://help-center.talenta.co/hc/id/articles/34576595122073-Bagaimana-Cara-Membuat-Job-Listing-pada-New-Talenta-Recruitment).
